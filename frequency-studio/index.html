How this works

Everything is synthesized live. There are no audio files here. Each preset defines a key, a chord progression, a tempo, and pattern rules per instrument. A scheduler walks a 16-step grid and asks each stem: do you play on this step, and what note? The Web Audio API then builds the sound from raw oscillators — the same subtractive-synthesis idea as a Moog or Juno: rich waveform → filter → envelope.

The whole studio is locked to A = 432 Hz. Every note frequency comes from f = 432 × 2^((midi−69)/12) — about 1.8% (31.8 cents) lower than standard 440 concert pitch, which many people describe as warmer and softer. Because everything is generated from this one reference, the entire arrangement is coherent: bass, pads, arps and plucks all agree.

The guitar is a physical model. It uses Karplus–Strong synthesis: a tiny burst of noise fed into a feedback delay line exactly one wave-period long, with a lowpass filter in the loop. The noise "plucks" the string; the loop sustains and darkens it, just like a real vibrating string losing high frequencies first.

The drone and binaural layers sit outside the tuning system on purpose. Solfeggio tones are fixed frequencies, so they're played pure. The binaural layer sends, e.g., 120 Hz to the left ear and 126 Hz to the right — the 6 Hz difference is perceived as a pulse inside your head, in the theta band associated with meditative, trance-like states.

The groove evolves. Several patterns roll dice each step (probability gates), and Intensity raises both velocity and the chance of extra hits — so the track breathes instead of looping identically.

Evolve mode lets the world play the studio. Time-of-day follows one smooth energy wave per day (trough at 3am, peak at 3pm). Weather comes from Open-Meteo — a free, keyless API — using your location if you allow it (an IP lookup as fallback): cloud cover darkens the filters, wind raises intensity, rain opens the reverb and shifts the scale to dorian, temperature bends the tempo. Sky mode needs no API at all: the moon phase and the sun's zodiac sign are computed from the date with real astronomy math — lunar illumination drives energy, and each sign transposes the key and colours the scale by its element. Every bar, the macros ease a few percent toward the data's target; every 16 bars the progression rotates. Nothing jumps — it drifts, like weather itself.
